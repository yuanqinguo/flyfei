# Go后端开发必备：Redis核心技术与实战
## 🚀 课程定位
- 专为**应届生/实习生/转行者**打造的Go后端Redis必修课
- 拒绝枯燥理论，直指实战与面试模拟。
- 10小时掌握Redis核心数据类型，1个综合项目串联全栈知识点
- 20+道精选面试题贯穿始终，让你从"会用"到"精通"，轻松应对技术面试。

---

## 🎯 课程特色

### **1. Go生态深度绑定**
- 全程基于**go-redis**客户端，代码100%可运行
- 演示Go协程+Redis的高并发模式
- 展示Go在Redis操作中的最佳实践
- “工程化实战型”的Redis+Go课程，而不是命令教学课

### **2. 四维学习模式**
每节课严格遵循：**核心原理 → 真实场景 → 手写代码 → 面试拷问**，确保知识被消化为实操能力

### **3. 面试题现场实战**
每讲穿插**2道高频面试题**，从Redis单线程原理到缓存雪崩解决方案，从分布式锁实现到Feed流架构设计，**题目即代码，代码即答案**

### **4. 大厂级综合项目**
第10讲混合使用**多中种Redis数据类型**，从零搭建**课程商品点赞关注系统**，产出可直接写进简历的完整项目

## 📚 课程大纲速览

| 课时 | 核心内容 | 实战案例 | 面试考点                       |
|------|---------|---------|----------------------------|
| **第1讲** | Redis架构与Go对接 | 环境搭建与连通性检测 | 单线程快的原因、连接池配置              |

| **第4讲** | List类型 | 异步任务队列 | 消息丢失问题、List vs Go channel  |
| **第5讲** | Set类型 | 文章标签系统 | "可能认识的人"实现、抽奖算法            |
| **第6讲** | Sorted Set | 游戏排行榜 | Score二次排序、千万级数据分页          |
| **第7讲** | Bitmap/HyperLogLog | 签到与DAU统计 | HyperLogLog误差、Bitmap连续签到   |
| **第8讲** | 持久化与事务 | 配置中心热更新 | RDB vs AOF选型、Redis事务局限性    |
| **第9讲** | 性能优化与架构 | Pipeline接口优化 | 缓存三大问题、Cluster哈希槽          |
| **第10讲** | **综合实战** | **课程商品系统** | **原子性设计、信息缓存优化**           |
