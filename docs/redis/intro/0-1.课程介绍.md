# 后端开发：Redis核心技术与实战

## 课程系列

1. 第一步：golang课程建设 [golang专题](https://yuanqinguo.github.io/flyfei/docs/golang/)
   1. go虚拟商品商城后台

   3. 【加基础课】后端必学：linux入门课
2. 第二步：Redis课程建设 [Redis专题](https://yuanqinguo.github.io/flyfei/docs/redis/index.md)

5. 第五步：微服务课程建设

## 课程定位
- 专为**应届生/初级后端/转行后端**打造的Go后端Redis必修课
- 拒绝枯燥理论，直指实战与面试模拟。
- 12-15节课掌握Redis核心数据类型，1个综合项目串联全栈知识点
- 结合真实场景贯穿始终，让你从"会用"到"熟练使用"，轻松应对技术面试。

---

## 课程特色

### **1. Go生态深度绑定**
- 全程基于**go-redis**客户端，代码100%可运行
- 展示Go在Redis操作中的最佳实践
- “工程化实战型”的Redis+Go课程，而不是命令教学课

### **2. 四维学习模式**
每节课严格遵循：**基础原理 → redis-cli操作 → 真实场景实战 → 总结**，确保知识被消化为实操能力

### **3. 综合项目**
第12讲混合使用**多中种Redis数据类型**，使用纯redis（无数据库），从零搭建**社交媒体点赞关注系统**
